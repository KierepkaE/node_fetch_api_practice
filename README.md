# node_fetch_api_practice

###### Simple command line app

- You can get information about a specific year that you choose.

#### How to use?

- clone this repo
- run `npm install`
- in the console: `node app.js 'year-that-you like' `
- example: `node app.js 2019`

#### Tech:

- `Node version: 10.15.0`
